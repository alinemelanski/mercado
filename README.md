# 💰 Controle de Finanças

Projeto inicial de um **Controle de Finanças** simples e divertido, feito em **HTML, CSS e JavaScript**.  
Objetivo: entregar ao usuário **facilidade e simplicidade** para organizar gastos e economizar dinheiro.

---

## 🚀 Primeira Etapa
- Estrutura básica criada com `index.html`, `style.css` e `script.js`.
- Formulário para adicionar gastos (descrição + valor).
- Lista dinâmica exibindo os gastos adicionados.
- Cálculo automático do **total**.
- Primeira animação com **GSAP**: moeda caindo ao adicionar gasto.

---

## 🛠️ Tecnologias usadas
- **HTML5** → estrutura da página  
- **CSS3** → estilos e layout  
- **JavaScript** → lógica de adicionar gastos e calcular total  
- **GSAP** → animações divertidas (moeda caindo)

## 🚀 Segunda Etapa
- Adição de categorias de gastos.
- Filtros para visualizar gastos por categoria.
- Totais acumulados por categoria.
- Persistência dos dados com LocalStorage.
- Interface aprimorada com CSS Grid/Flexbox.

## terceira Etapa
- Planejamento de gráficos interativos.
